PHP Y MySQL

Introducción:

MySQL es un sistema gestor de bases de datos relacionables que es de código abierto. En palabras mas simples, es un gestor que nos ayuda a organizar la información en diferentes secciones de manera estructurada.

Conexión entre php y MySQLi.

Para realizar este proceso es necesario que nosotros primero, conozcamos la dimensión de nuestra base de datos y el tipo de contenido que vamos estar manipulando. Es por eso que es recomendable, escribir en papel las posibles tablas que se van a insertar.

En primera instancia debemos de crear una carpeta que va a contener nuestro programa o app dentro de nuestro servidor web. Hecha esa carpeta, dentro de ella se deberán agregar cuatro carpetas mas.

  • Model o Modelo: Esta carpeta contiene la interacción o su conexión con la base de datos.
  • Controllers o Controladores: Esta carpeta contiene toda la configuración que nuestra aplicación va a obtener, desde dar de alta registros, consultas, guardado, eliminación, etc.
  • Views o Vistas: Las vistas son las paginas con las que el usuario estará interactuando. Aquí es donde podemos encontrar la estructura html combinada con php.
  • Public o Publica: Esta carpeta contiene los archivos “css”, “img” y “js” que la aplicación / programa va a mostrar en pantalla.

Controlador que registrara a los usuarios.

<?php 
    /*
    
    Dentro de los controlares desarrollaremos la logica necesaria para nuestra app
    o programa que estemos desarrollando, en este caso, sera un controlador que nos
    ayudara a registrar a los usuarios que sean ingresados mediante el formulario.
    
    */
    
    /*
    Primero incluimos la conexion a la base de datos, por que tenemos que verificar SI
    los campos tenian contenido y SI el boton fue oprimido.
    */
    include("../model/conn_db.php");
    
    /*
        Verificamos SI el boton fue oprimido por el usuario. Si, si fue, ENTONCES,
        checaremos que los campos esten llenos y no vacios.
    */
    if (isset($_POST['register'])) {
            /*
                Primero verificamos si dentro de los campos de texto existe al menos mas 1
                letra dentro de ellos, es por eso que decimos, SI dentro de el campo "name" Y
                campo "email" hay mas de una palabra. ENTONCES se realizara la siguiente
                instruccion.
            */
        if (strlen($_POST['name']) >= 1 && strlen($_POST['email']) >= 1) {
    
                    /*
                        Tomaremos los valores que se hayan insertado dentro de los campos
                        (imputs). Y checaremos con la funcion trim que la cadena de texto
                        no contenga espacios al inicio y al final de este.
                    */
            $name = trim($_POST['name']);
            $email = trim($_POST['email']);
                    
                    /*
                        Usaremos la funcion Date para mostrar dentro de la base de datos
                        la fecha en la que el usuario se dio de alta dentro de la base de datos
                        al momento de mandar el formulario.
                    */
            $fechareg = date("d/m/y");
            $consulta = "INSERT INTO usuarios(nombre, correo) VALUES ('$name','$email')";
            $resultado = mysqli_query($conn_db, $consulta);
            if ($resultado) {
                ?> 
                <h3 class="ok">¡Te has inscripto correctamente!</h3>
               <?php
            } else {
                ?> 
                <h3 class="bad">¡Ups ha ocurrido un error!</h3>
               <?php
            }
        }   else {
                ?> 
                <h3 class="bad">¡Por favor complete los campos!</h3>
               <?php
        }
    }
    
    ?>

mysqli_query en PHP es una función que se utiliza para enviar una consulta SQL a una base de datos MySQL y obtener un resultado
